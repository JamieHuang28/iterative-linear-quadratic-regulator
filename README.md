# iLOR(iterative linear quadratic regulator)
iLOR(iterative linear quadratic regulator) is a method for solving optimal control problems. It is a variant of the iLQR(iterative linear quadratic regulator) method, which is a method for solving optimal control problems.
This project is designed by huangzhengming28@gmail.com and implemented by GPT agent.

## Specification
* Language: Python3.10
* Dynamic System Involved: Object Rotation Control by apply torque to a rigid body
* lqr.py: Implementation of the LQR algorithm, which is specified in linear-quadratic-regulator.tex
* ilqr.py: Implementation of the iLQR algorithm, which is specified in iterative-linear-quadratic-regulator.tex
* prompt-xxx.md: Prompt log for xxx Agent
* implement_request_prompt.md: Code implementation request prompt for any GPT Agent for experiment.
